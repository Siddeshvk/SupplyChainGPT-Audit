# SupplyChainGPT-Audit

**Open-source AI tool that scans Bills of Lading for Incoterms errors, multilingual compliance failures, and regulatory risks.**

Built by Sid Vithal (Siddesh Vithal Kudachikar) in 2026 to democratize supply-chain AI for 300,000+ SMEs and defense suppliers.

## Live Demo
https://supplychaingpt.streamlit.app 

## Features
- Extracts Incoterms and checks consistency
- Consensus-driven multilingual validation
- Regulatory flags (customs, CMMC implications)
- Zero cost, no login, works on any device
